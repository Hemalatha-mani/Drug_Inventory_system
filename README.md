# Drug_Inventory_system
Drug inventory and supply chain tracking system using React, Spring Boot, and PostgreSQL

# Project Title :
Drug Inventory and Supply Chain Tracking System

# Problem Statement :

Many hospitals and medical institutions face challenges such as:

Drug shortages
Overstocking of medicines
Lack of real-time inventory visibility
Delayed deliveries from suppliers
Expired medicines remaining in stock
Manual record keeping

A centralized system is needed to track medicines from procurement to final consumption.

# Project Objectives :

Maintain accurate medicine inventory records.
Track medicine procurement and distribution.
Monitor supplier/vendor activities.
Generate alerts for low stock and expiry dates.
Analyze medicine consumption patterns.
Provide dashboard-based monitoring.
Improve efficiency in the drug supply chain.
